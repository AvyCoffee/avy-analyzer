# Avy Coffee | Roast Analyzer v4.5.1

This is the official deployment repository for the **Avy Coffee Roast Analyzer**, a specialized tool for coffee professionals to track roast profiles, manage bean inventory, and calculate production margins.

## 🚀 Version 4.5.1 Highlights
* **Managerial Cost Tracking:** Integrated labor, utility, packaging, and depreciation math.
* **Customer CRM:** Dedicated database for tracking wholesale and retail client profiles.
* **Sensory Lab:** Post-roast quality control with 0-100 scoring and cupping notes.
* **Inventory Management:** Real-time green bean stock tracking and automated deductions.
